# ⏰ NativeLoopTimer (原生多任务定时中心)

> 🚀 **拒绝 200MB 内存的臃肿！** 专为 Windows 设计的「极轻量、超低功耗、绝对免打扰」原生多任务倒计时与智能闹钟效率利器。

---

## 🎯 为什么我们需要它？（击中痛点的“避坑指南”）

目前市面上大部分的定时器或效率工具，都存在以下让人抓狂的**致命缺陷**：

* ❌ **臃肿的 Electron 架构**：区区一个定时器，动辄打包出 **100MB+** 的体积，运行时吞噬 **200MB+** 内存！让轻薄本续航缩水，老电脑风扇狂转。
* ❌ **霸道夺取输入焦点**：你正在全屏打联机游戏、疯狂敲代码或者做 PPT，倒计时突然到期，弹窗瞬间抢走了你的鼠标和键盘光标！游戏被迫弹回桌面，正在输入的文字直接中断，体验极差。
* ❌ **假死与“睡眠遗忘症”**：电脑一旦合盖进入休眠，很多软件的计时器就直接“假死”挂掉。醒来后错过的会议、吃药提醒全部被遗漏，根本没有补发机制。
* ❌ **任务栏与托盘重影**：手抖多次双击了图标，任务栏直接多出五六个一模一样的重复进程，关也关不完，乱成一团。

## ✨ 它是如何“拿捏”你的使用体验的？（五大杀手级特性）

### 1. 🍃 极度克制的“低功耗原生架构”
* 拒绝套壳浏览器！采用 Python 原生 Win32 消息循环和 CustomTkinter 精心雕琢。
* 运行内存仅需 **几 MB**，CPU 占用趋近于 **0%**。单文件直接双击运行，没有多余的依赖垃圾。

### 2. 🤫 绝对的免打扰通知 (Zero Focus Stealing)
* 深度调用 Windows 原生 Toast 消息推送机制，配合优化的顶层绘制，通知只会在右下角静默划出。
* **绝对不会抢走你的当前焦点**。哪怕你正在全屏激烈枪战或打字，你的输入流也完全不会被中断！如果不小心误触了通知，程序也会在后台静默继续，毫无干扰。

### 3. 🛌 聪明的“休眠唤醒补偿” (Power-State Smart Compensation)
* 深度集成 Windows 电源管理，监听系统级休眠与唤醒事件（`WM_POWERBROADCAST`）。
* 当电脑合盖休眠、再次醒来时，系统会**立刻判定并毫秒级补发**所有休眠期间错过的提醒，并以唤醒时间为基准，智能重跑新一轮倒计时。

### 4. 🎹 极致高效的物理输入框 (Physical Auto-Tabbing)
* 闹钟输入采用**物理分割的小时、分钟独立输入域**。
* **极速盲打体验**：输入满 2 位数自动跳转到下一格，敲击方向键或退格键实现无缝回退，输入效率比臃肿的传统下拉滑动选择框高出十倍！

### 5. 🔒 天生防呆：单实例互斥锁与“自动唤醒” (Single-Instance Interceptor)
* 内置本地回路互斥 Socket 锁。多次双击运行本软件？没关系！
* 新进程探测到旧进程在运行后，会**向老进程发送唤醒电波，直接把已有实例从后台托盘叫醒、浮现并获取焦点**，然后新进程自动静默消亡，保持你的托盘区永远只有一个清爽干净的图标！

## 🎨 现代高颜值界面 (Harmonious Modern Aesthetics)

* **极客暗黑风**：基于精挑细选的和谐暗黑 HSL 色彩（深灰、幽蓝、薄荷绿与琥珀金组合），长时间盯盘不伤眼。
* **自适应滚动容器**：任务卡片无限添加？界面底层采用全局滚动布局，右侧的滚动条只有在内容溢出时才会悄悄浮现，未溢出时自动消失，视界纯粹而极致。
* **常驻后台托盘**：点击“开始”后，主窗口彻底消失，只在右下角系统托盘常驻一个小图标，支持右键直接全局暂停、恢复和进入设置。

## 📦 如何下载与使用？

### 🚀 极速启动（推荐）
1. 前往本仓库的 [Releases](https://github.com/thisisfrankryan/NativeLoopTimer/releases) 页面。
2. 下载打包好的单文件可执行程序 `NativeLoopTimer.exe`。
3. 双击直接运行即可，无需安装，无广告，无多余依赖。

### 💻 开发者自编译
如果你希望基于源码自行修改和编译：
```bash
# 1. 克隆仓库
git clone https://github.com/thisisfrankryan/NativeLoopTimer.git
cd NativeLoopTimer

# 2. 安装核心依赖
pip install customtkinter pystray winrt-Windows.UI.Notifications winrt-Windows.Data.Xml.Dom pywin32

# 3. 运行项目
python main.py

# 4. 自行打包 exe
pyinstaller NativeLoopTimer.spec --noconfirm
```
